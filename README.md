## 📦 Setup

```bash
npm install
npm run dev
```

## 🔑 Get Free Gemini API Key

1. Go to [aistudio.google.com](https://aistudio.google.com)
2. Click **Get API Key**
3. Copy and paste in ZARA's input screen

## 🏗️ Build & Deploy

```bash
npm run build
# Upload dist/ folder to Vercel or Netlify
```

## 📱 Install as Mobile App (PWA)

1. Open deployed URL in Chrome (Android)
2. Tap ⋮ → **Add to Home Screen**
3. ZARA icon appears on your home screen!
